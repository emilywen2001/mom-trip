# 妈妈的旅行小助手 · 启动说明

## 项目结构

```
mom-travel/
├── src/           前端代码 (React + TypeScript)
├── backend/       后端代码 (Python FastAPI)
├── dist/          前端构建产物
└── 启动说明.md
```

## 启动步骤

### 第一步：启动后端

打开一个终端，进入 backend 目录：

```bash
cd backend
python main.py
```

后端运行在 http://localhost:8000
API 健康检查: http://localhost:8000/api/v1/health

### 第二步：启动前端（开发模式）

再开一个终端：

```bash
npm run dev
```

前端运行在 http://localhost:5173

### 环境变量配置（backend/.env）

```
GLM_API_KEY=你的智谱key
OPENWEATHER_API_KEY=你的OpenWeatherMap key（可选，没有用mock数据）
```

## OpenWeatherMap 免费 Key 获取

1. 访问 https://openweathermap.org/api
2. 注册账号 → My API Keys
3. 复制 Default key 填入 .env

（没有 Key 也能运行，天气用 Mock 数据）

## 演示流程（5分钟）

1. **首页** - 查看行程进度、天气、今日穿搭
2. **行程页** - 查看每日时间轴，或 AI 规划新旅行
3. **助手页** - 问 AI 旅行问题，流式回答
4. **拍照页** - 开启相机拍照，查看相册
5. **我的页** - 查看统计、安全设置
